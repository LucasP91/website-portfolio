# Visual Hierarchy

Arranging elements so the eye reads them in order of importance. The single most important UI quality: a user should grasp "what matters most" in under a second.

## Core idea

Every screen has a **priority order**. Make the most important thing the most prominent, the second most important second, and so on. Avoid "everything is bold" — if everything shouts, nothing is heard.

## Tools for creating emphasis (strongest → subtlest)

- **Size / scale** — bigger = more important. Headlines dominate; captions recede.
- **Weight** — bold vs. regular pulls focus.
- **Color & contrast** — a saturated/high-contrast element jumps forward; muted tones recede.
- **Position** — top and left get seen first in LTR reading; the optical center draws the eye.
- **Whitespace** — isolating an element with space makes it feel important.
- **Density / grouping** — proximity implies relationship (Gestalt).

## Reading patterns

- **F-pattern** — text-heavy pages (articles, search results): users scan top, then down the left edge. Front-load key words.
- **Z-pattern** — sparse, visual pages (landing pages): eye sweeps top-left → top-right → diagonal → bottom. Place logo, CTA, and final action along the Z.

## Rules of thumb

- **One primary action per view.** A single, visually dominant CTA. Secondary actions are visibly lower-weight (outline/ghost/text button).
- **3 levels of text hierarchy** are usually enough: heading, subhead/body, supporting/caption.
- **Contrast creates hierarchy** — vary size *and* weight *and* color together for clear steps, not 12 nearly-identical sizes.
- **Group then separate** — cluster related controls; use space or dividers between groups.
- **Limit competing focal points** — two equally loud elements fight; pick a winner.

## Checklist

- [ ] Is there exactly one clear focal point / primary action?
- [ ] Can you squint at the screen and still tell what's most important?
- [ ] Do heading levels step down clearly (size + weight)?
- [ ] Are related items grouped and unrelated items separated?
- [ ] Are secondary/tertiary actions visually subordinate to the primary one?
